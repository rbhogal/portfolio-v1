Challenges

Being my first project I took away many lessons as there were many firsts. The greatest challenge was simply making the page responsive. I had set too many hard widths and heights and therefore got many horizontal scrollbars when I tried to make the page responsive. So I learned to save a lot of time and headache, code with responsiveness in mind from the very beginning.

I also had good experience and practice to learn and pick things on spot when called for as I had never made an accordion or email validation form before.
